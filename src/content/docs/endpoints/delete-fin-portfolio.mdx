title: Delete Financial Portfolio
---

The Delete Financial Portfolio Delete endpoint allows you to delete an existing portfolio.

## Endpoint

---

- `DELETE` - /financial-portfolio/{`portfolio_id`}

## Request

---

This endpoint contains the following arguments:

**Headers**

- `x-rapidapi-key` (string, required): Your API key
- `x-rapidapi-host` (string, required): The server URL, excluding the protocol part of the URL, "https://".

**Parameters**

- `portfolio_id` (float, required): The unique identifier of the portfolio to be updated.

## Response

---

The response includes the following attributes, which provide details about the data returned.

**Metadata**

- `status` (string): Indicates the current status of the API response. Values include include "success" or "error".

- `message` (string): A human-readable message indicating the result of the request.

**Response data**

The `portfolio` object provides the below response attribute:

---

- `deletion_time` (string): The time when the portfolio was deleted in UTC.
