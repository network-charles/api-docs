title: Get Currency Exchange Rate
---

The Currency Exchange Rate endpoint displays the exchange rate between two currencies.

## Endpoint

---

- `GET` - /currency-exchange-rate

## Request

---

This endpoint contains the following arguments:

**Headers**

- `x-api-key` (string, required): Your API key
- `x-api-host` (string, required): The server URL, excluding the protocol part of the URL, "https://".

**Parameters**

- `from_symbol` (string, required): The currency you are converting from.
- `to_symbol` (string, required): The currency you are converting to.
- `language` (string): Your preferred language

## Response

---

The response includes the following attributes, which provide details about the data returned.

**Metadata**

- `status` (string): Indicates the current status of the API response. Values include include "success" or "error".

- `request_id` (string): A unique identifier for the API request. This can be used for tracking and debugging purposes, allowing you to trace specific requests in logs or support inquiries.

**Response data**

The `data` object includes the following response attributes:

---

- `from_symbol` (string): The currency you are converting from.
- `to_symbol` (string): The currency you are converting to.
- `type` (string) The type of currency
- `exchange_rate` (float): The current exchange rate after comparing both currencies.
- `previous_close` (float): The previous close.
- `last_update_utc` (integer): The time this query was made.
