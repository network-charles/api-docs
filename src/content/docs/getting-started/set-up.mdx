title: Enviroment Setup

This guides you on how to set up the development for interacting with this API
locally.

import { Steps } from "@astrojs/starlight/components";

<Steps>

1. Prerequisites

   - **Python (3.7 or later)**: You can download Python from the [official documentation](https://www.python.org/downloads/).
   - **Pip3**: Python comes pre-installed with Pip3, a package manager.

2. Create a Virtual Enviroment

   ```bash
   python -m venv venv

   source venv/bin/activate  # macOS/Linux

   venv\Scripts\activate  # Windows
   ```

3. Install the Flask Library

   ```python
   pip3 install flask
   ```

4. Create your API Key

   ```bash
   export API_KEY=1234567890
   nano ~/.zshrc # For macOS
   source ~/.zshrc # Apply changes

   nano ~/.bashrc  # For most Linux distributions
   source ~/.bashrc # Apply changes

   nano ~/.profile  # For compatibility in some environments
   source ~/.profile # Apply changes
   ```

5. Create the Flask App

   ```bash
   wget https://raw.githubusercontent.com/network-charles/api-docs/refs/heads/main/src/content/docs/getting-started/mock-api.py
   python3 mock-api.py   # Append & so it runs in the background
   ```

6. Verify your API Key

   ```bash
   curl -X GET http://127.0.0.1:5000/get-api-key
   ```

</Steps>
